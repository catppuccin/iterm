## Usage

1. Launch iTerm 2
2. Type CMD+i (âŒ˜+i)
3. Navigate to the **Colors** tab
4. Click on **Color Presets**
5. Click on **Import**
6. Select the **catppuccin.itermcolors**
7. Click on **Color Presets** and choose catppuccin
8. Enjoy! :sparkles:
